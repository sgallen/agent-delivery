## How to use these

Do not adopt every playbook at once.

Start with one low-risk project or one safe class of work. Build the smallest version of the system that can produce trustworthy evidence. Then improve it from what the run teaches you.

The operating model should grow from reality, not from an afternoon of enthusiastic diagramming.
