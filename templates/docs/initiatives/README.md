# Initiatives

Store durable initiative investment records here.

Suggested naming:

```text
INIT-004-audit-workflows.md
```

Start from `../INITIATIVE.md`.

An initiative record should preserve:

- value hypothesis and baseline;
- decisions and stop conditions;
- staged resource and Change-funnel forecasts;
- linked Change dispositions, landing states, and actuals;
- released or non-release outcome;
- delivery, resolution, and value variance;
- scheduled value or decision-value reviews.

Do not use an initiative record as a second backlog. Link the underlying Changes and summarize what their evidence means for the investment.
